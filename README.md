# Rowell Family Reunion Website

A static website for the Rowell Family Reunion event.

## Structure

- `index.html` — Landing page with event details and RSVP form
- `styles.css` — Stylesheet
- `script.js` — RSVP form logic and localStorage persistence

## Features

- Event information landing page
- RSVP form with client-side validation
- Data persists in browser localStorage (survives page refresh)
- Edit/update previously submitted RSVP

## Usage

Open `index.html` in any browser. No build tools or server required.
